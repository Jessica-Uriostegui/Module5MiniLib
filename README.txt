Library is a command line interface.
It performs various operations adding books, 
checking out books, returning books, 
viewing details of authors and users

Run main.py Follow main menu instruction.
Adding book : It will ask if author and genre have been 
added already.
If not you will need to add them first before adding a book.
If you do not have a user ID you will also be asked to add user.
After making all selection you can exit.
